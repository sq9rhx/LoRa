| File         | Address | 
| :--------------- | :------- | 
| bootloader.bin   | 0x1000       |
| partitions.bin        | 0x8000       | 
| firmware.bin         | 0x10000      |
| spiffs.bin | 0x3D0000        |

If you want update only without lost configuration you can upload firmware.bin
