## Pins (SX1262)
all pins used by the **UEXT** connector

| Function         | GPIO Pin | 
| :--------------- | :------- | 
| SPI Clock        | 14       |
| SPI MISO         | 15       | 
| SPI MOSI         | 2       | 
| Chip Select (CS) | 5        |
| Radio Reset      | 4       | 
| Interrupt / DIO1 | 13       |
| Busy Signal      | 36       | 
| Receive Enable   | -1       |
| Transmit Enable  | 16       |
