-   **Radio:** SX1268 Ebyte E22-400M30S radio chip.
-   **LoRa Power:** 1W, TCXO

## Pins (SX1268)
almost all pins used by the **UEXT** connector (exception (GPIO33))

| Function         | GPIO Pin | 
| :--------------- | :------- | 
| SPI Clock        | 14       |
| SPI MISO         | 15       | 
| SPI MOSI         | 2       | 
| Chip Select (CS) | 5        |
| Radio Reset      | 4       | 
| Interrupt / DIO1 | 33       |
| Busy Signal      | 36       | 
| Receive Enable   | 13       |
| Transmit Enable  | 16       | 
| OLED SDA   | 32       |
| OLED SCL  | 0       |
